# Image Assets

Place your images in this folder:

- `profile.jpg` or `profile.png` - Your profile photo
- `project-1.jpg` - Screenshot for AI Credit Scoring Model
- `project-2.jpg` - Screenshot for Sales Forecasting project
- `project-3.jpg` - Screenshot for Credit Portfolio Dashboard

## Recommended Image Sizes

- Profile photo: 400x400px (square)
- Project images: 800x600px (landscape)

## Image Optimization

For best performance, optimize your images before uploading:
- Use WebP format when possible
- Compress images to reduce file size
- Use appropriate dimensions (don't upload oversized images)
