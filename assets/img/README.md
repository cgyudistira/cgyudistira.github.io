# Image Assets

Place your images in this folder:

## Logo & Favicon

- `logo.png` or `logo.svg` - Website logo (appears in navbar and footer)
  - Recommended size: 150x40px or similar aspect ratio
  - Format: PNG with transparent background or SVG
  - Will auto-resize to fit navbar (max height: 40px)
  
- `favicon.png` - Browser tab icon
  - Recommended size: 32x32px or 64x64px
  - Format: PNG or ICO
  - Will be used for browser tab, bookmarks, and mobile home screen

## How to Replace

1. **Logo**: Replace `logo.png` or `logo.svg` with your own logo file (keep the same filename)
2. **Favicon**: Replace `favicon.png` with your own icon (keep the same filename)
3. The website will automatically use your new images without code changes

## Project Images (Optional)

- `profile.jpg` or `profile.png` - Your profile photo
- `project-1.jpg` - Screenshot for AI Credit Scoring Model
- `project-2.jpg` - Screenshot for Sales Forecasting project
- `project-3.jpg` - Screenshot for Credit Portfolio Dashboard

## Recommended Image Sizes

- Logo: 150x40px (landscape) or 40x40px (square)
- Favicon: 32x32px or 64x64px (square)
- Profile photo: 400x400px (square)
- Project images: 800x600px (landscape)

## Image Optimization

For best performance, optimize your images before uploading:
- Use WebP or PNG format for logos with transparency
- Use SVG for scalable logos (best quality at any size)
- Compress images to reduce file size
- Use appropriate dimensions (don't upload oversized images)
